About This App

This is a simple task management application that helps you organize your daily tasks.

Features:

  • Add new tasks with titles and descriptions
  • Set due dates and times
  • Set reminders for important tasks
  • Edit and delete existing tasks
  • View all your tasks in one place

How to Use:

  1. Fill in the task title (required)
  2. Add a description if needed
  3. Set a due date and time
  4. Check "Set Reminder" and choose a reminder time if you want notifications
  5. Click "Add Task" to save
